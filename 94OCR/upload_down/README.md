
- flask项目使用，必须创建名字对应的文件，否则无法找到，
  - static文件
      存放要使用的js和css文件，单独创建一个 css 或者 js 文件夹，
      
  - templates文件夹
  
      存放要加载的html文件
  
  - 如下：
  
    ```python
    static
    	css
        	css1.css
            css2.css
    	js
        	jquery-3.6.0.min.js
    templates
    	index.html
    main.py
    ```
